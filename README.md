# Netflix Movies and TV Shows Dashboard

## Project Overview

This Tableau project analyzes Netflix movies and TV shows using an interactive dashboard. The dashboard shows content distribution by type, country, release year, rating, director, and title details.

## Dashboard Link

View the interactive dashboard on Tableau Public:

https://public.tableau.com/app/profile/vibhashree.g.m3399/viz/NetflixAnalysis_16264517674090/DashboardNetflix

## Dataset

Dataset: Netflix Movies and TV Shows  
Source: Kaggle  

https://www.kaggle.com/datasets/shivamb/netflix-shows

The dataset includes Netflix title information such as type, title, director, country, date added, release year, rating, duration, and listed category.

## Tools Used

- Tableau
- Kaggle Dataset
- Data Visualization
- Dashboard Design

## Dashboard Features

- Movie and TV show count
- Netflix titles by country
- Movies and TV shows by year
- Top 10 directors
- Rating-wise title distribution
- Detailed movie and TV show list

## Key Insights

- The dashboard shows 5,377 movies and 2,410 TV shows.
- Movies are higher in number than TV shows.
- Netflix content is available across many countries.
- More titles were added or released in recent years.
- TV-MA appears as one of the most common rating categories.
- The dashboard helps quickly explore Netflix content patterns.
